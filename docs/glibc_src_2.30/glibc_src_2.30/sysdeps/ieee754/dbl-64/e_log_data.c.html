55/* Algorithm:
56
57 x = 2^k z
58 log(x) = k ln2 + log(c) + log(z/c)
59 log(z/c) = poly(z/c - 1)
60
61where z is in [1.6p-1; 1.6p0] which is split into N subintervals and z falls
62into the ith one, then table entries are computed as
63
64 tab[i].invc = 1/c
65 tab[i].logc = (double)log(c)
66 tab2[i].chi = (double)c
67 tab2[i].clo = (double)(c - (double)c)
68
69where c is near the center of the subinterval and is chosen by trying +-2^29
70floating point invc candidates around 1/center and selecting one for which
71
72 1) the rounding error in 0x1.8p9 + logc is 0,
73 2) the rounding error in z - chi - clo is < 0x1p-66 and
74 3) the rounding error in (double)log(c) is minimized (< 0x1p-66).
75
76Note: 1) ensures that k*ln2hi + logc can be computed without rounding error,
772) ensures that z/c - 1 can be computed as (z - chi - clo)*invc with close to
78a single rounding error when there is no fast fma for z*invc - 1, 3) ensures
79that logc + poly(z/c - 1) has small error, however near x == 1 when
80|log(x)| < 0x1p-4, this is not enough so that is special cased. */
