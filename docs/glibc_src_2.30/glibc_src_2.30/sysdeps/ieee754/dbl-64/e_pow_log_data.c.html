40/* Algorithm:
41
42 x = 2^k z
43 log(x) = k ln2 + log(c) + log(z/c)
44 log(z/c) = poly(z/c - 1)
45
46where z is in [0x1.69555p-1; 0x1.69555p0] which is split into N subintervals
47and z falls into the ith one, then table entries are computed as
48
49 tab[i].invc = 1/c
50 tab[i].logc = round(0x1p43*log(c))/0x1p43
51 tab[i].logctail = (double)(log(c) - logc)
52
53where c is chosen near the center of the subinterval such that 1/c has only a
54few precision bits so z/c - 1 is exactly representible as double:
55
56 1/c = center < 1 ? round(N/center)/N : round(2*N/center)/N/2
57
58Note: |z/c - 1| < 1/N for the chosen c, |log(c) - logc - logctail| < 0x1p-97,
59the last few bits of logc are rounded away so k*ln2hi + logc has no rounding
60error and the interval for z is selected such that near x == 1, where log(x)
61is tiny, large cancellation error is avoided in logc + poly(z/c - 1). */
