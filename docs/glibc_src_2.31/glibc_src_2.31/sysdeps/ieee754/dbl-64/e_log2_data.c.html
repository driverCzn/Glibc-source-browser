56/* Algorithm:
57
58 x = 2^k z
59 log2(x) = k + log2(c) + log2(z/c)
60 log2(z/c) = poly(z/c - 1)
61
62where z is in [1.6p-1; 1.6p0] which is split into N subintervals and z falls
63into the ith one, then table entries are computed as
64
65 tab[i].invc = 1/c
66 tab[i].logc = (double)log2(c)
67 tab2[i].chi = (double)c
68 tab2[i].clo = (double)(c - (double)c)
69
70where c is near the center of the subinterval and is chosen by trying +-2^29
71floating point invc candidates around 1/center and selecting one for which
72
73 1) the rounding error in 0x1.8p10 + logc is 0,
74 2) the rounding error in z - chi - clo is < 0x1p-64 and
75 3) the rounding error in (double)log2(c) is minimized (< 0x1p-68).
76
77Note: 1) ensures that k + logc can be computed without rounding error, 2)
78ensures that z/c - 1 can be computed as (z - chi - clo)*invc with close to a
79single rounding error when there is no fast fma for z*invc - 1, 3) ensures
80that logc + poly(z/c - 1) has small error, however near x == 1 when
81|log2(x)| < 0x1p-4, this is not enough so that is special cased. */
